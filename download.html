Install Steps

  1. Download Online DAW macOS.dmg.
  2. Open the DMG.
  3. Drag Online DAW.app into Applications.
  4. Open the app from Applications.
  5. If macOS warns the first time, approve it in Privacy & Security and open it again.

Support

  • If the app does not open, confirm you downloaded the full DMG and moved the app into Applications.
  • If you want the browser version instead, you can still run python3 server.py and open http://127.0.0.1:8000.
  • If a plugin is detected but does not open its editor, that means the local plugin bridge still needs a native host command configured for that plugin workflow.
